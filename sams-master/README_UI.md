# 山姆会员商店自动抢购系统 - Web UI版本

## ✨ 新增功能

本项目现在支持**可视化Web界面**，提供更友好的用户体验！

## 🚀 快速开始

### 方式一：Web UI模式（推荐）

```bash
# 1. 安装依赖
go get github.com/gorilla/websocket
go mod tidy

# 2. 启动Web服务器
go run . server

# 3. 打开浏览器访问
# http://localhost:8080
```

### 方式二：命令行模式（原版）

```bash
go run main.go --authToken=xxxxx
```

## 📸 界面预览

### 主要功能

1. **配置面板**：可视化配置所有参数
2. **流程监控**：实时显示执行步骤
3. **商品列表**：显示购物车中的商品
4. **配送时间**：显示可用时间段
5. **实时日志**：所有操作日志实时显示
6. **订单信息**：抢购成功后显示订单详情

### 界面特点

- 🎨 现代化设计，美观易用
- 📱 响应式布局，支持移动端
- 🔄 实时更新，无需刷新
- 📊 状态可视化，一目了然
- 🎯 操作简单，一键开始

## 🔄 工作流程

```
配置参数 → 保存配置 → 开始抢购 → 实时监控 → 完成支付
```

## 📋 功能对比

| 功能 | 命令行模式 | Web UI模式 |
|------|-----------|-----------|
| 参数配置 | 命令行参数 | 可视化表单 |
| 状态显示 | 终端输出 | 实时界面 |
| 日志查看 | 终端滚动 | 日志面板 |
| 商品列表 | 文本输出 | 卡片展示 |
| 配送时间 | 文本输出 | 列表展示 |
| 订单信息 | 文本输出 | 卡片展示 |
| 操作方式 | 命令行 | 点击按钮 |

## 🎯 使用场景

### Web UI模式适合：
