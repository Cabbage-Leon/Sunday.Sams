# 🚀 快速开始指南

## 第一步：安装依赖

```bash
cd sams-master
go get github.com/gorilla/websocket
go mod tidy
```

## 第二步：启动服务器

### Windows用户
双击运行 `start_server.bat`

或使用命令行：
```bash
go run . server
```

### Linux/Mac用户
```bash
chmod +x start_server.sh
./start_server.sh
```

或使用命令行：
```bash
go run . server
```

## 第三步：打开浏览器

访问：**http://localhost:8080**

## 第四步：配置参数

1. 填写 **Auth Token**（必填）
   - 从山姆APP的HTTP请求头中获取
   - 这是最重要的参数

2. 填写其他可选参数
   - 地址ID、配送类型、支付方式等

3. 点击 **"保存配置"** 按钮

## 第五步：开始抢购

1. 确认配置已保存（会显示收货地址）
2. 点击 **"开始抢购"** 按钮
3. 实时监控执行流程
4. 等待抢购成功

## 常见问题

### Q: 如何获取Auth Token？

A: 使用抓包工具（如Charles、Fiddler）或浏览器开发者工具，查看山姆APP的HTTP请求头中的 `auth-token` 字段。

### Q: 端口被占用怎么办？

A: 可以指定其他端口：
```bash
go run . server 8081
```
然后访问 `http://localhost:8081`

### Q: 无法连接服务器？

A: 
1. 检查Go环境是否正确安装
2. 检查端口是否被占用
3. 检查防火墙设置

### Q: 配置保存失败？

A:
1. 检查authToken是否正确
2. 检查网络连接
3. 查看日志面板的错误信息
